Extension

A Concordion extension adds additional functionality to Concordion, for example custom commands and listeners.

To use extensions, set the system property concordion.extensions to a comma-separated list of the fully-qualified class name of the extensions to be installed. All extensions must be present on the classpath and must implement org.concordion.api.extension.ConcordionExtension.

Extensions are able to:

  • Add custom commands to Concordion
  • Act as listeners to Concordion commands and processing events
  • Create resources in the Concordion output folder
  • Add CSS or JavaScript to the Concordion output
