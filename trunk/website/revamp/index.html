Key Benefits

  • Powerful, yet simple to use Integrates directly with JUnit.

  • Highly readable tests Concordion acceptance tests not only work well, but read well too. This means the tests double up as documentation of system behaviour, and, since they're linked to the system, they're always in sync.

  • Separates tests from implementation Tests that include a lot of implementation detail lock you into that implementation. Concordion helps you to document the logic and behaviour of your system in a way that does not lock you in.
