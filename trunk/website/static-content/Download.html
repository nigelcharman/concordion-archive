Latest Release Candidate: 1.3.1-RC3   (01-Mar-2009)

  • concordion-1.3.1-RC3.zip (full distribution including source code and all dependencies)
  • Requires: Java 1.5 or above
  • New command: concordion:run (contributed by Stein Kåre Skytteren) lets you link to another specification and run it, displaying the link's background in green / red appropriately. You might use this to create a page containing a list of all the acceptance tests for a feature, in a similar way you would use a JUnit test suite.
  • Upgraded to support JUnit 4.5.
  • Special variable "#HREF" provides access to href attribute for a link. This can be used when you want to reference test data from a specification. (Example)
  • Please feedback any questions / issues to the mailing list.

Latest Stable Version: 1.2.0

  • concordion-1.2.0.zip (full distribution including source code and all dependencies)
  • Requires: Java 1.5 or above
  • New command: concordion:assertTrue

  • JUnit 4.4 example: 
    package spec.examples;

import org.concordion.integration.junit4.ConcordionRunner;
import org.junit.runner.RunWith;

@RunWith(ConcordionRunner.class)
public class Demo {

    public String greetingFor(String firstName) {
        return String.format("Hello %s!", firstName);
    }
}
  • Support for JUnit 4.4 @BeforeClass, @Before, @After, @AfterClass annotations.
